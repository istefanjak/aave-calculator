This is a simple AAVE staking rewards tracker and calculator. Data is being fetched from EtherScan API automatically every 6 seconds while the app is being open.

Usage:
  • Default total daily reward is set to 400. It can be changed by entering a different ammount in the input box. A different ammount can also be initialized by typing the number preceded by '/' after the app URL. For example https://istefanjak.github.io/aave-calculator/600 will init the app with 600 daily reward.
  • Total AAVE staked shows current AAVE staked pool in real time
  • If you don't have tokens staked yet, enter that ammount in the first input box
  • If you have tokens currently staked in the pool enter that ammount in the second input box

If you received "Too many requests" error:
  • The app is being accessed from multiple devices/ browsers/ tabs on same IP address. This wont work.
  • You just manually refreshed/ revisited the page while 6 seconds haven't passed yet. Just wait for the next refresh in 6 seconds and it should work.
