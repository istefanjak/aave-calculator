This is a simple AAVE staking rewards tracker and calculator. All data is being fetched from Etherscan servers. To see how old is the fetched data, check the 'Time data was fetched' label in the calculator.

Usage:
  • Default daily emission is being fetched from the server. To use a different emission for calculation purposes, check the 'Use custom emission' checkbox. Enter desired value and confirm.
  • Total AAVE staked shows current AAVE staked pool in real time
  • If you don't have tokens staked yet, enter that amount in the first input box
  • If you have tokens currently staked in the pool enter that amount in the second input box
